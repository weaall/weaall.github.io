Generic이란?


컴파일을 하는 시점에 타입을 지정해줄 수 있는 기능

장점


  • 코드의 재사용성 증가
  • 타입 안정성 확보
  • 유연한 타입 적용 가능
  • 컴파일시 타입을 체크하여 타입 오류를 방지

기본 문법

  • < T > 를 이용해 함수가 호출될 때 전달된 배열의 요소 타입을 자동으로 결정
