## Math.ceil()

```java
for (int i = 0; i < n; i++) {
    array[i] = (int) Math.ceil((100.0 - progresses[i]) / speeds[i]);
}
```

## Math.min() / Math.max() 

```java
int[][] sizes // 이차원 배열

int length = 0, height = 0;

for (int[] card : sizes) { // 이차원 배열 순회
    length = Math.max(length, Math.max(card[0], card[1])); // 두값중 큰 값을 length에 넣음
    height = Math.max(height, Math.min(card[0], card[1])); // 두값중 작은 값을 height에 넣음
}
```
